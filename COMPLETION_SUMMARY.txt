🎨 USER WORKFLOW
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. UPLOAD STEP FILE
     User visits /upload
       ↓ Drag-drop or select STEP file
       ↓ Real-time progress tracking
       ↓ Auto-redirect to 3D viewer

  2. EXPLORE 3D MODEL
     View at /viewer/{modelId}
       ↓ Rotate, zoom, pan model
       ↓ Interactive orbit controls
       ↓ IKEA-style isometric view

  3. REVIEW PARTS
     Navigate to /parts/{modelId}
       ↓ See all extracted parts
       ↓ View classifications
       ↓ Check dimensions & volume

  4. FOLLOW ASSEMBLY
     Go to /assembly/{modelId}
       ↓ Read step-by-step instructions
       ↓ See AI-generated details
       ↓ Review tips & warnings
       ↓ Export to PDF


🚀 QUICK START

  $ npm install
  $ export NEXT_PUBLIC_API_URL=http://localhost:8000/api/v1
  $ npm run dev
  
  → Open http://localhost:3000
  → Upload a STEP file
  → Enjoy!
